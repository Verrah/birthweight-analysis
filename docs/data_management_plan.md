\# Data Management and Sharing Plan

\## NIH Format | Birthweight Analysis Project



\---



\## 1. Types of Data



\- NCHS Natality Detail Files 1969–1971 (public-use)

\- NVSS Bridged-Race Population Estimates

\- Derived analytical datasets (county-level aggregates)

\- Analysis code (Python, R)

\- Model outputs and visualizations



\---



\## 2. Data Standards



\- Fixed-width ASCII following NCHS specifications

\- Race/ethnicity coded to NVSS bridged-race categories

\- Missing data represented as NA with codebook documentation

\- Geographic identifiers follow FIPS standards



\---



\## 3. Repository and Archiving



\- \*\*Code:\*\* GitHub (https://github.com/Verrah/birthweight-analysis)

\- \*\*Data:\*\* Open Science Framework (OSF) — aggregate outputs only

\- \*\*Archive:\*\* ICPSR deposit within 6 months of publication

\- \*\*Format:\*\* CSV (data) + plain-text codebooks



\---



\## 4. Access and Restrictions



\- Public-use data and derived aggregates: CC-BY 4.0

\- Restricted-use NCHS files: accessible via NCHS RDC only

\- Cell suppression: counts <10 suppressed per NCHS protocols



\---
